Inside a Computer

  • CPU (Central Processing Unit): The brain of the computer that executes instructions and performs calculations.
  • RAM (Random Access Memory): Temporary storage used by the computer for actively running programs and data.
  • Storage Devices: Hard Disk Drives (HDDs) or Solid State Drives (SSDs) used for permanent data storage.
  • Motherboard: Main circuit board connecting all components together.
  • GPU (Graphics Processing Unit): Responsible for rendering graphics and images on the display.
  • PSU (Power Supply Unit): Converts AC power from the outlet into DC power for the computer's components.
  • Cooling Systems: Fans and heat sinks to keep components cool and prevent overheating.
